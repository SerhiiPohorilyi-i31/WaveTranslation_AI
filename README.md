Репозиторій для практичних та самостійних робіт.
Інформаційна система - веб-перекладач WaveTranslation

## Як запускати Matllab з Fuzzy Logic Toolbox
1. Перейти за посиланням SWI-Prolog https://matlab.mathworks.com/
2. у командному рядку MATLAB ввести: fuzzyLogicDesigner
3. Імпортувати файл .fis

## Як запускати Prolog
1. Встанови SWI-Prolog https://www.swi-prolog.org
2. `swipl`
3. `?- [prolog/wavetrans].`
4. Приклад запиту: `?- good_translation("Добрий день", T).`
